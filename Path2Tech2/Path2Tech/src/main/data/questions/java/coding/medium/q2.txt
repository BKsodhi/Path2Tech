Question: Write a program to reverse a string.

Description:
- Write a program that takes a string as input and returns the string reversed.
- Do not use any built-in reverse functions.

Input:
- A single string s (1 <= |s| <= 1000).

Output:
- The reversed string.

Examples:
Input: "hello"
Output: "olleh"

Input: "world"
Output: "dlrow"
